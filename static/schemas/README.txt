TEI XML Schemas for Saga-Scribe
================================

tei_all.xsd
-----------
TEI P5 (Text Encoding Initiative, version P5) complete schema.
Source: https://www.tei-c.org/release/xml/tei/custom/schema/xsd/tei_all.xsd
License: BSD-2-Clause and Creative Commons Attribution 4.0 International

The TEI Guidelines are maintained by the TEI Consortium:
https://tei-c.org/

Additional schemas can be placed in this directory and will be
automatically detected by Saga-Scribe's validation system.
